- **Cloud Application Programming Model (CAP):** [cap-js](https://github.com/cap-js) · [cap-java](https://github.com/cap-java) · [cap-js-community](https://github.com/cap-js-community) · [capire](https://github.com/capire)

### Foundations

SAP is a founding member of the [NeoNephos Foundation](https://neonephos.org/), which fosters collaboration on open source cloud-native technologies. Explore the foundation's [projects](https://neonephos.org/projects) to see how SAP contributes to shaping the future of enterprise cloud infrastructure.
